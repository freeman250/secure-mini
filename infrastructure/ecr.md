# AWS ECR Push

The Docker image for the Flask application was built in AWS Cloud9
and pushed manually to AWS Elastic Container Registry (ECR).

Repository:
541373007083.dkr.ecr.us-east-1.amazonaws.com/secureme-mini

Steps:
1. Build image using build.sh
2. Authenticate Docker to ECR
3. Tag image for ECR
4. Push image to ECR
